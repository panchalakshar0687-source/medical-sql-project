# Medical SQL Project

This is a SQL project based on medical data stored in CSV files.

## About the Project
This project uses SQL to answer a set of questions using medical datasets.
The main goal is to practice writing SQL queries.

## Project Structure
- data/ – Contains all CSV data files
- sql/ – Contains SQL queries that answer the project questions
- report/ – Contains the project report in Word format

## Data
The data includes information about patients, doctors, admissions, and locations.
These CSV files are used to create tables in the database.

## SQL Queries
All queries are written in one SQL file.
Each query is written to answer one question.

## Tools Used
- SQL
- CSV files

## Summary
- This project shows how SQL can be used to work with data
- Queries use joins, filters, and basic calculations

## Challenges
- Understanding how the tables are related
- Writing correct SQL queries
